About me

I'm a Python developer at the start of a journey I'm genuinely excited about. Right now my focus is the foundation: control flow, functions, string manipulation, and the standard library — learned properly, not skimmed. I practice daily in Jupyter Notebook and Spyder, and I push what I learn to GitHub so my progress stays public and honest.

Where I'm headed: data work and building real, useful tools. I'm leveling up steadily — one notebook, one commit, one concept at a time.
